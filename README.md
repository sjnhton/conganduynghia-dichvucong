# Landing Page Công an xã Duy Nghĩa

Website landing page chính thức của Công an xã Duy Nghĩa, thành phố Đà Nẵng.

## Giới thiệu

Đây là website cung cấp thông tin về các dịch vụ công trực tuyến và kênh phản ánh khó khăn, vướng mắc cho người dân.

## Tính năng

- ✅ **Hero Section**: Logo Công an nhân dân, thông tin liên hệ, slogan
- ✅ **Dịch vụ công trực tuyến**: 3 dịch vụ với đường link đến Cổng dịch vụ công quốc gia
- ✅ **Form phản ánh**: Đăng ký phản ánh khó khăn, vướng mắc
- ✅ **Responsive**: Tương thích hoàn hảo trên mọi thiết bị
- ✅ **Animations**: Hiệu ứng chuyển động mượt mà

## Cách chạy

### Cách 1: Mở trực tiếp
1. Mở file `index.html` bằng trình duyệt web (Chrome, Firefox, Edge...)

### Cách 2: Sử dụng Live Server (VS Code)
1. Cài đặt extension **Live Server** trong VS Code
2. Nhấp chuột phải vào file `index.html`
3. Chọn **Open with Live Server**

### Cách 3: Sử dụng Python HTTP Server
```bash
# Python 3
python -m http.server 8000

# Truy cập: http://localhost:8000
```

### Cách 4: Sử dụng Node.js
```bash
# Cài đặt http-server
npm install -g http-server

# Chạy server
http-server

# Truy cập: http://localhost:8080
```
